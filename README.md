# 💰 용돈 기록 (Minseo5)

문자(카드 승인 알림)를 그대로 가져와 **자동으로 지출 내역을 정리**해 주는 앱입니다.
직접 타이핑할 필요 없이, 문자를 공유하거나 복사해서 붙여넣기만 하면 됩니다.

---

## 📱 메인 화면 한눈에 보기

```
┌─────────────────────────────────────┐
│  용돈 기록                    ⋮ 메뉴 │   ← 제목 / 내보내기·가져오기
├─────────────────────────────────────┤
│   ◀      2026년 06월       ▶         │   ← 이전달 / 이번달 / 다음달
│   합계: 114,600원                    │   ← 이번 달 총 지출
├─────────────────────────────────────┤
│  카카오T_바이크            950원     │
│  2026-06-10 07:07                    │   ← 사용처 / 금액 / 날짜·시각
│  ─────────────────────────────────  │
│  드림디포수지점          20,000원    │
│  2026-06-08 16:25                    │
│  ─────────────────────────────────  │
│  배스킨라빈스             9,800원    │
│  2026-06-08 22:31                    │
│                                      │
│                              ┌─────┐ │
│                              │  ➕  │ │   ← 추가 버튼 (FAB)
│                              └─────┘ │
└─────────────────────────────────────┘
```

## ➕ 지출 추가하기 (3가지 방법)

오른쪽 아래 **➕ 버튼**을 누르면 메뉴가 나옵니다.

```
┌──────────────────────────┐
│  추가 방법 선택           │
│                          │
│  📨 문자에서 가져오기     │   ← ① 문자앱을 열어 공유
│  📋 복사된 항목 가져오기  │   ← ② 복사해 둔 문자 붙여넣기
│  ✏️  직접 입력            │   ← ③ 손으로 직접 입력
└──────────────────────────┘
```

### ① 📨 문자에서 가져오기 (가장 많이 씀)

```
 [➕] → [문자에서 가져오기]
        │
        ▼
 ┌──────────────┐      문자앱이 열립니다
 │   문자앱     │      메시지를 길게 눌러
 │  (삼성메시지)│  →   "공유" → "용돈 기록" 선택
 └──────────────┘
        │
        ▼
 ┌──────────────┐
 │ 파싱 화면    │      문자 내용이 자동으로 채워짐
 └──────────────┘      → [파싱 시작] 누르기
```

### ② 📋 복사된 항목 가져오기

문자나 메모에서 텍스트를 **복사(길게 눌러 복사)** 한 뒤,
`➕ → 복사된 항목 가져오기` 를 누르면 파싱 화면에 그 내용이 들어옵니다.
(이후 동작은 문자 공유와 똑같습니다.)

### ③ ✏️ 직접 입력

날짜 / 금액 / 용도를 손으로 입력합니다.

```
┌──────────────────────────┐
│  지출 추가                │
│  사용날짜 [2026-06-12  ]  │
│  금액     [        5000]  │
│  용도     [점심          ]│
│            [취소]  [저장] │
└──────────────────────────┘
```

## 🔍 파싱 화면 사용법 (핵심!)

문자를 가져오면 이 화면이 나옵니다.

```
┌─────────────────────────────────────┐
│  ← 문자에서 가져오기                  │
├─────────────────────────────────────┤
│  문자 내용                            │
│  ┌─────────────────────────────────┐│
│  │[Web발신] 하나2*8*체크승인        ││   ← 가져온 문자 원문
│  │차*서 950원 06/10 07:07 카카오T   ││     (여러 건 들어와도 OK)
│  └─────────────────────────────────┘│
│  ┌─────────────────────────────────┐│
│  │           파싱 시작              ││   ← ① 먼저 이 버튼!
│  └─────────────────────────────────┘│
└─────────────────────────────────────┘
```

**[파싱 시작]** 을 누르면 ↓ 아래에 결과가 나옵니다.

```
│  파싱 결과 3건 (중복 1건)            │
│  ┌─────────────────────────────────┐│
│  │ 카카오T_바이크          950원   ││  ← 정상 (흰 배경)
│  │ 2026-06-10 07:07                ││
│  ├─────────────────────────────────┤│
│  │ [중복] 드림디포        20,000원 ││  🟥 연한 빨강 = 이미 있는 항목
│  │ 2026-06-08 16:25                ││     (적용 시 자동 제외)
│  └─────────────────────────────────┘│
│                                      │
│  🔵 ⚠ 파싱 안 된 항목 1건:           │  🔵 파란 글씨 = 규칙에 안 맞아
│     • 아빠 나 초계국수 먹고싶으       │     자동 인식 못한 부분
│                                      │
│  합계: 20,950원                      │  ← 전체 합계
│  중복 제외(적용 대상): 950원         │  ← 실제로 추가될 금액
│  ┌─────────────────────────────────┐│
│  │              적용               ││  ← ② 확인했으면 적용!
│  └─────────────────────────────────┘│
```

### 🎨 색깔 의미 (아주 중요!)

| 색깔 | 뜻 | 어떻게 되나요? |
|---|---|---|
| ⬜ 흰 배경 | 정상 인식된 지출 | 적용하면 저장됨 |
| 🟥 **연한 빨강** | **이미 저장된 것과 중복** | 적용해도 **자동으로 빠짐** (중복 안 생김) |
| 🔵 **파란 글씨** | **자동 인식이 안 된 부분** | 무시되거나, 직접 입력으로 추가 |

> 💡 **중복 판단 기준**: 같은 **날짜 + 시각(시·분) + 사용처 + 금액** 이면 같은 지출로 봅니다.
> 그래서 같은 문자를 두 번 가져와도 두 번 저장되지 않습니다.

**[적용]** 을 누르면 중복을 뺀 항목만 저장되고, 자동으로 그 거래가 있는 **달 화면으로 이동**합니다.

---

## 🗓️ 월 이동 / 이번 달 합계

```
   ◀      2026년 06월       ▶
   합계: 114,600원
```

- **◀ / ▶** : 이전 달 / 다음 달로 이동
- **합계** : 그 달의 총 지출 금액

---

## ✅ 여러 개 골라서 합계 보기 (선택 모드)

특정 항목들만 골라 합계를 보고 싶을 때 씁니다.

```
 1) 항목을 길게 누르면          2) 다른 항목을 톡톡 누르면
    선택 모드 시작                 체크(파란 배경)됩니다
 ┌──────────────────────┐      ┌──────────────────────┐
 │ ← 2개 선택      합계 │      │ ← 3개 선택      합계 │
 ├──────────────────────┤      ├──────────────────────┤
 │🟦 카카오T     950원  │      │🟦 카카오T     950원  │
 │🟦 드림디포  20,000원 │      │🟦 드림디포  20,000원 │
 │   베라       9,800원 │      │🟦 베라       9,800원 │
 └──────────────────────┘      └──────────────────────┘
                                         │
                          [합계] 버튼 누르면 ▼
                       ┌──────────────────────────┐
                       │ 선택 합계: 30,750원 (3건)│
                       │  카카오T          950원  │
                       │  드림디포       20,000원 │
                       │  베라            9,800원 │
                       │                  [닫기]  │
                       └──────────────────────────┘
```

- **길게 누르기** → 선택 모드 시작
- **톡 누르기** → 선택 / 해제
- 위쪽 **[합계]** → 고른 항목들의 합계
- **←** 또는 **뒤로가기** → 선택 모드 끝내기

---

## ✏️ 항목 수정 / 삭제

일반 상태에서 항목을 **톡 누르면** 수정 창이 열립니다.

```
┌──────────────────────────┐
│  수정                     │
│  사용날짜 [2026-06-10  ]  │
│  금액     [         950]  │
│  용도     [카카오T_바이크]│
│  [삭제]        [취소][저장]│   ← 삭제도 여기서
└──────────────────────────┘
```

---

## 💾 데이터 내보내기 / 가져오기

오른쪽 위 **⋮ 메뉴**:

- **내보내기** : 모든 기록을 JSON 파일로 저장 (백업용)
- **가져오기** : 저장해 둔 JSON 파일에서 불러오기 (중복은 자동 제외)

가져온 문자 원문은 자동으로 `내장저장소/Documents/Minseo5/data.json` 에도 쌓입니다.

---

## ⚙️ 파싱 규칙 바꾸기 (고급)

문자 자동 인식 규칙은 **소스 수정 없이** 아래 파일만 고치면 됩니다.

```
내장저장소 / Documents / Minseo5 / rules.json
```

정규식으로 "어디가 금액이고 어디가 날짜·사용처인지"를 정해 줍니다.
새 카드/은행 문자 형식이 생기면 이 파일에 규칙을 추가하세요.
(앱을 다시 켜면 적용됩니다.)

> 🔒 파일 접근을 위해 앱 첫 실행 시 **"모든 파일 액세스"** 권한을 허용해 주세요.
> 허용하지 않으면 기본 규칙(하나 체크카드)만 동작합니다.
