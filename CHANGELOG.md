# Changelog

이 파일은 Minseo5(용돈 기록) 앱의 주요 변경 사항을 기록합니다.

## [1.3.0.0] - 2026-06-12

### Added
- **소비 트렌드 화면(통계)**: 메뉴 → "통계"에서 선택 월의 카테고리 도넛 + 일별 지출 막대 + 사용처 TOP5 + 합계/전월 대비를 한눈에 (MPAndroidChart)
- **카테고리 자동 분류**: 사용처 텍스트를 키워드로 분류. 분류 규칙은 외부 파일 `Documents/Minseo5/categories.json`로 분리해 소스 수정 없이 편집 가능 (파싱 룰과 동일한 버전 기반 시드 방식). 표시 시점 계산이라 룰만 고치면 과거 기록도 자동 재분류
- 기록 없는 달 빈 상태(🐷 + "기록 추가하러 가기")와 "기타" 비중 과다 시 룰 보강 힌트

### Changed
- 시안 디자인 적용: 메인/통계 화면을 Material You 톤(녹색 요약 카드 + 둥근 카드 + 파스텔 원형 카테고리 타일)으로 통일. 리스트 금액은 빨강, 같은 카테고리는 리스트·통계에서 동일 색
- 상태바/네비바 아이콘 색을 배경 밝기에 맞춰 자동 설정(라이트=어두운 아이콘, 다크=밝은 아이콘)으로 가독성 확보

### Fixed
- 메인 메뉴(통계/내보내기/가져오기)가 표시되지 않던 문제 (`setSupportActionBar`와 Toolbar 자체 메뉴 충돌 제거)
- 다크모드에서 차트 텍스트·축·범례가 안 보이던 문제 (테마 색 기반으로 변경)

## [1.2.1.0] - 2026-06-12

### Changed
- 앱 아이콘을 전용 디자인(동전 든 아이 + 돼지저금통)으로 교체 (적응형 아이콘 + 밀도별 PNG)

## [1.2.0.0] - 2026-06-12

### Added
- 중복 체크: 같은 날짜+시각(시·분)+사용처+금액이면 중복으로 보고, 파싱 화면에서 **연한 빨강 배경**으로 표시하고 적용 시 자동 제외
- 파싱 결과 하단에 **합계 / 중복 제외(적용 대상) 금액** 표시
- 파싱 안 된 블록을 **파란색 리스트**로 표시 ("어느 부분이 인식 안 됐는지")
- 거래 시각(HH:mm) 저장 및 표시
- 메인 리스트 **선택 모드**: 길게 눌러 선택 → 항목 체크 → 상단 "합계"로 선택 항목 합계 보기
- 아주 쉬운 사용법 README (화면 도식 포함)
- MinseoStore 배포 등록 (APK 이름 지정 + scripts/apk.sh)

### Changed
- 하단 navigation bar / 상단 status bar에 리스트가 겹치지 않도록 레이아웃 수정 (CoordinatorLayout → LinearLayout)

### Fixed
- 같은 문자를 여러 번 가져와도 중복 저장되던 문제

## [1.1.1.0] - 2026-06-12

### Added
- "추가 방법 선택"에 "복사된 항목 가져오기" 추가: 클립보드에 복사한 문자/텍스트를 문자 공유와 동일하게 파싱 화면으로 가져옴

## [1.1.0.0] - 2026-06-12

### Added
- 문자 앱에서 공유로 지출 내역 가져오기: 문자 앱을 열어 메시지를 공유하면 앱으로 텍스트가 전달됨 (READ_SMS 권한 불필요)
- 파싱 화면: 가져온 문자 표시 → "파싱 시작"으로 결과(날짜/금액/용도) 확인 → "적용"으로 일괄 DB 저장
- 한 문자에 여러 건이 들어있어도 모두 추출 (find-all)
- 외부 룰 파일 `Documents/Minseo5/rules.json`: 정규식 기반 파싱 규칙을 소스 수정 없이 편집. 앱 시작 시 읽고, 없으면 기본 룰(assets) 자동 생성
- 가져온 문자 원문을 `Documents/Minseo5/data.json`에 누적 저장
- 적용 후 해당 거래 월로 화면 자동 이동

### Changed
- 상단 status bar / 하단 navigation bar를 가리지 않도록 시스템 인셋 처리
- SMS 목록 UI를 카드형으로 정리 (이후 공유 방식 전환으로 목록 화면은 파싱 화면으로 대체)

### Fixed
- 입력 다이얼로그 진입 시 `AlertDialog` 캐스팅 크래시 (`android.app.AlertDialog` → `androidx.appcompat.app.AlertDialog`)
- 연도 없는 카드 문자(MM/DD)의 날짜를 가장 최근 과거 기준으로 추론

## [1.0.0.0] - 2026-06-11

### Added
- 용돈 기록 앱 초기 구현: 월별 지출 목록, 이전/다음달 이동, 직접 입력, JSON 내보내기/가져오기, Room DB 저장
